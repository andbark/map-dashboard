School Photography Client Map Dashboard

Static Version Instructions

This is the static (offline) version of the Map Dashboard application. You can run it locally on your computer without needing any internet connection or Vercel account.

How to Use:

  1. Extract the zip file (map-dashboard-static.zip) to a folder on your computer.
  2. Open the index.html file in your web browser (Chrome, Firefox, Safari, or Edge).
  3. Use the dashboard to:
    • Upload CSV files with school data
    • Map columns if your CSV has different header names
    • View schools on the interactive map
    • Save data for future use

Note: Since this is running locally in your browser, all data will be saved to your browser's local storage. This means:

  • Data is only accessible on the device and browser you're using
  • Clearing browser data will erase any saved schools
  • You can export/save your data if needed

Features:

  • Geocoding: Automatically converts addresses to map coordinates
  • Column Mapping: Allows flexibility with CSV columns that don't match standard names
  • Local Storage: Saves imported schools data for future sessions
  • Interactive Map: View and interact with school locations

Enjoy using your Map Dashboard!
